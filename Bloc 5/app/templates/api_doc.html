Welcome to Wine quality prediction API !

This API allows you to use a machine learning model trained on wine data to predict its quality score, which is between 0 and 10. The data used are :

  • Fixed acidity : tartaric acid - g / dm^3
  • Volatile acidity : acetic acid - g / dm^3
  • Citric acid - g / dm^3
  • Residual sugar - g / dm^3
  • Chlorides : sodium chloride - g / dm^3
  • Free sulfur dioxide - mg / dm^3
  • Total sulfur dioxide - mg / dm^3
  • Density - g / cm^3
  • PH - level of acidity
  • Sulphates : potassium sulphate - g / dm3
  • Alcohol - % by volume

How to use the API

API is located at https://wineapp1002.herokuapp.com/predict/ . The model can take a single input or multiple ones. You can use it those ways :

  • With python :
    You can use a python script to make the API requests. To do so you can use 'requests' library.

    Example with simple input:
    import requests
    url = "https://wineapp1002.herokuapp.com/predict/"
    input_simple = { "input": [[7.0, 0.27, 0.36, 20.7, 0.045, 45.0, 170.0, 1.001, 3.0, 0.45, 8.8]] }
    res = requests.post(url, json=input_simple)
    res.json()
    Output: ['predict : ', [6]]

    Example with multiple inputs:
    import requests
    url = "https://wineapp1002.herokuapp.com/predict/"
    input_multiple = { "input": [[7.0, 0.27, 0.36, 20.7, 0.045, 45.0, 170.0, 1.001, 3.0, 0.45, 8.8], [5.0, 0.98, 0.32, 18.9, 0.050, 75.0, 122.0, 0.401, 3.1, 0.21, 1.2]] }
    res = requests.post(url, json=input_multiple )
    res.json()
    Output: ['predict : ', [6, 5]]

  • With shell :
    You can use the "curl" command like this :

    Example with simple input:
    curl -i -H "Content-Type: application/json" -X POST -d '{"input": [[7.0, 0.27, 0.36, 20.7, 0.045, 45.0, 170.0, 1.001, 3.0, 0.45, 8.8]]}' https://wineapp1002.herokuapp.com/predict/
    Output: ['predict : ', [6]]

    Example with multiple inputs:
    curl -i -H "Content-Type: application/json" -X POST -d '{ "input": [[7.0, 0.27, 0.36, 20.7, 0.045, 45.0, 170.0, 1.001, 3.0, 0.45, 8.8], [5.0, 0.98, 0.32, 18.9, 0.050, 75.0, 122.0, 0.401, 3.1, 0.21, 1.2]]}' https://wineapp1002.herokuapp.com/predict/
    Output: ['predict : ', [6, 5]]
